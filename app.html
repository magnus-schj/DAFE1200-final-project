OUR APP

THE ULTIMATE REWARD FOR SORTING GARBAGE!

The Home Page

The app has an easy-to-use interface. The home page displays a map where the nearest recycling station is located. The map also displays where the user is located so they can easily find the fastest route to our station. At the bottom screen, there are three buttons. The left most button guides the user to their profile page. The middle button brings the user back to the homepage. While on the homepage, when this button is pressed, a search-bar pops-up. With this the user can find different recycling stations that are not located nearby. The right button guides the user to our event list.

Shop

There are also two buttons located in the top right corner of the homepage. The button located at the very top guides the user to our shop page where they can use their accumulated points on discounts or lottery tickets. The lottery tickets gives the user a chance to win prizes that are handed out weekly, monthly or yearly.

Events

At the event list page, the user can see which recycling events that are located near them. By clicking on one of the events, they can also see the exact location of the event and when the event is about to take place.

Friends

At the profile page, the top right button brings out a friendslist window. Here the user has an option to add more friends and the option to see their friends accumulated points.

Statistics

At the profile page, there is another button that brings out a statistics window. These statistics tells the user how much they have recycled in one day, one week, one month or throughout the entire year. The user can also see what type of waste they have recycled the most.
